My Completed Projects

Minesweeper

This is my first solo project. I chose this one so that I could learn how to use 2-dimensional arrays.

I also chose to challenge myself by not using any hardcoded HTML and used elements created in JS since I knew that I would be using react later and thought that it would help with keeping track of pieces of information while thinking.

Lessons learned:

I learned that I should definitely figure out how to approach the problem I have in concept before I start working on it on my computer. I began mixing up a lot of my code and functionalities so I decided that maybe I should tackle on bigger concepts and go down to the very fine details. This project taught me how to use the debugger in chrome.

Technologies Used

  • HTML
  • CSS
  • JS
Stock Market View

This project was the first project I completed with React. I wanted to learn how to use a lot of numerical data and then display them on screen in a way that anyone could understand without reading through 100 lines of numbers.
